# Al-Buraq Website

A simple logistics website with an online booking system.

## Project Overview
This website allows customers to submit booking requests for logistics and transportation services.
The main focus of the project is simplicity and ease of use for customers.

## Features
- Multi-page website (Home, About, Services, Vehicles, Booking, Contact)
- Online booking form
- Contact page
- Responsive navigation menu
- Data storage using PHP and MySQL

## Booking Process
1. The user fills in the booking form.
2. The form submits data to `save_booking.php`.
3. PHP validates the input and stores it in the MySQL database.
4. The booking appears in the `bookings` table in phpMyAdmin.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
